# Security Policy

## Supported Versions

Currently, only the latest version of Links & More is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.3.x  | :white_check_mark: |
| < v1.3  | :x:                |

## Reporting a Vulnerability

We take the security of Links & More seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, please report it through one of the following methods:

1.  **Email**: Please email [INSERT CONTACT EMAIL] with a description of the vulnerability.
2.  **GitHub Private Reporting**: If enabled on this repository, use the "Security" tab to "Report a vulnerability".

### What to include in your report

*   A description of the vulnerability.
*   Instructions on how to reproduce the issue.
*   Potential impact of the vulnerability.

### What to expect

*   You will receive an acknowledgment of your report within 48 hours.
*   We will provide an estimated timeline for a fix and keep you informed of our progress.
*   Once the issue is resolved, we will post a security advisory and credit you for the discovery (if desired).
